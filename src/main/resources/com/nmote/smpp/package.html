Main package for Nmote SMPP 5.0 API. This package contains all SMPP abstractions.

Session represents a communication context for executing SMPP commands.
Link transports SMPP PDUs between sessions.
Command packs request and response PDUs in a single executable object.
*PDU represents various SMPP PDUs (Protocol Data Units)
Processor executes Commands
SMPPTime, SMPPDistributionList, SMPPAddress represents various SMPP related values

For more information see Nmote SMPP 5.0.doc file
